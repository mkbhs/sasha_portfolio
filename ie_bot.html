Indoor Explorer Robot

Differential Drive Mobile Robot suitable for Indoor Exploration

Teleoperation Mode

A basic visual interface has been implemented to control the motion of the ie_bot. The visual information is captured by the robot and sent to the teleoperation interface. At the same time the user can sent motion commands from the interface to the mobile robot. All the communication is done via tcp/ip (wireless) using the local WLAN.
The mobile robot uses as actuators two DC motors connected to a dual-half bridge that is managed by the microcontroller. The computer handles the wireless communication, the visual capturing system. There is a battery for the computer and another different battery for the motors.
